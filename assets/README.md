# Asset Requirements

For Microsoft Store submission, you need the following image assets:

## Required Logos

### Square 150x150 Logo
- **File**: `Square150x150Logo.png`
- **Size**: 150x150 pixels
- **Used for**: Medium tile on Start menu

### Square 44x44 Logo
- **File**: `Square44x44Logo.png`
- **Size**: 44x44 pixels
- **Used for**: App list icon, taskbar

### Wide 310x150 Logo
- **File**: `Wide310x150Logo.png`
- **Size**: 310x150 pixels
- **Used for**: Wide tile on Start menu

### Store Logo
- **File**: `StoreLogo.png`
- **Size**: 50x50 pixels
- **Used for**: Microsoft Store listing

## Asset Guidelines

- Use PNG format with transparent background
- Follow Windows 11 design guidelines
- Avoid text in logos (use symbol/icon only)
- Ensure good contrast for both light and dark themes
- Consider using vector graphics and exporting at required sizes

## Creating Assets

You can use the official 0 A.D. logo assets:
1. Download from: https://trac.wildfiregames.com/
2. Resize to required dimensions
3. Export as PNG with transparency
4. Place in this directory
