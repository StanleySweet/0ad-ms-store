1. Visit the 0 A.D. website: https://play0ad.com/
2. Download official logo/icon assets from the game
3. Resize to required dimensions
4. Replace files in this directory

### Option 2: Create Custom Assets
Design assets following Windows 11 guidelines:
- Use the 0 A.D. game icon/symbol
- Transparent or themed background
- Good contrast for light/dark modes
- No text in logos (icon/symbol only)
